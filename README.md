# Sistema de Cadastro de Aluno em Java

Este projeto foi desenvolvido como parte do meu aprendizado em Java, com foco nos conceitos fundamentais de Programação Orientada a Objetos (POO).

---

## Sobre o projeto

Venho da linguagem de Programação Python, da qual aprendi um pouco de lógica de programação. Percebi grandes diferenças entre Python e o Java mas não me deixei desanimar e continuei tentando.

Este projeto é um **instrumento de estudos**, criado para reforçar conceitos importantes e consolidar minha base na linguagem Java.

---

## Funcionalidades

* Cadastro de nome e idade do aluno
* Inserção de notas
* Cálculo de média
* Verificação do status do aluno:

  * Excelente
  * Aprovado
  * Reprovado
* Menu interativo no terminal

---

## Conceitos aplicados

* Classes e Objetos
* Encapsulamento (getters e setters)
* Métodos com retorno
* Separação de responsabilidades
* Estruturas de decisão (if/else)
* Estrutura de repetição (do-while)

---

## Tecnologias utilizadas

* Java
* IntelliJ IDEA

---

## Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/java-estudos-poo.git
```

2. Abra o projeto no IntelliJ IDEA

3. Execute a classe `Main`
